# Technical / CEO Questions Checklist (HiringBand / Junior iOS)

- [ ] Який поточний технологічний стек iOS-проєкту (мінімальна iOS версія, архітектура, dependency strategy)?
- [ ] Наскільки критичним є WebKit/WKWebView у продукті та які головні ризики цього шару?
- [ ] Які сторонні SDK вже інтегровані (аналітика, push, трекінг, ad services) і як контролюється їхня стабільність?
- [ ] Як реалізовані deeplink / universal link сценарії для користувачів і для модерації App Store?
- [ ] Який CI/CD pipeline використовується зараз і що планується покращити?
- [ ] Хто відповідає за сертифікати, provisioning profiles і push keys, та як організовано доступи?
- [ ] Який процес підготовки релізу й обробки фідбеку після App Review?
- [ ] Які типові причини інцидентів у production і як влаштований postmortem-процес?
- [ ] Які очікування до якості коду, code review та тестового покриття в команді?
- [ ] Які продуктові цілі на найближчі 6–12 місяців і яку роль у них відіграє iOS-напрям?

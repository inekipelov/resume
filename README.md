## Overview

- Markdown under `resumes/<year>/<role>/<language>/<variant>/resume.md` is the only source of truth.
- PDF files are generated from the `release` branch and published to `resume-pdfs-latest`.
- Current published exports cover the 2025 iOS Developer and 2026 Software Engineer resumes.
